# 🍊 Vantage Intelligence by Modgroup

> **Versione 1.0.0 - Marzo 2026** > Strategia, Trasparenza e Tecnologia per l'Hospitality di Lusso.

---

## 🚀 Vision
Vantage è lo strumento di **Market Intelligence** proprietario di Modgroup.it, progettato per offrire ai direttori d'hotel una visione in tempo reale delle proprie performance rispetto ai competitor, con un focus specifico su:
* **Rate Parity Control** (Analisi disparità tariffarie).
* **Ad Library Spying** (Monitoraggio campagne Google/Meta dei competitor).
* **Vantage Index** (Punteggio di salute digitale dell'hotel).

## 🛠️ Tech Stack
L'app è costruita con le tecnologie web più avanzate del 2026:
* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS 4.0](https://tailwindcss.com/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Animations:** Framer Motion
